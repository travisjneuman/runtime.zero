# runtime.zero Brand Assets

This folder contains repository-safe copies of owner-provided `runtime.zero` / `rz0` visual candidates.

The original working assets remain outside the repository and must not be edited in place. If an asset needs cropping, resizing, conversion, favicon generation, Open Graph export, README treatment, or TUI/CLI splash adaptation, create a derivative copy in a new folder such as `assets/brand/derived/<date>/` and document it here.
